## Desarrollo

1- Clonar repositorio

```bash
git clone https://github.com/martinfyic/frontend-agencia.git
```

2- Dirigirse al directorio de proyecto descargado:

```bash
cd frontend-agencia/
```

3- Ejecutar el comando:

```bash
pnpm install && pnpm dev
```
